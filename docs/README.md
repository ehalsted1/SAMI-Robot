# Social Animated Mechanical Interlocutor (SAMI) Robot Overview and Documentation
The Social Animated Mechanical Interlocutor (SAMI) robot is made up of several pieces. The physical robot itself, a high-level controlling computer or microcontroller, and the sensor and interaction interface box.

## What is SAMI?
SAMI is a stationary social robot, originally designed as a platform for in-situ, long-term research with older adults. The original 3D model design of the SAMI body is based on the [Aster robot](https://www.thingiverse.com/thing:3992150), which is itself based on the [Poppy robot](https://www.poppy-project.org/en/).
